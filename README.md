## Development

### Run the project

```
$ git clone git@github.com:UMN-Blockchain/UMN-Blockchain-Website.git
$ cd gatsby-netlify-cms
$ yarn
$ yarn develop
```

To test the CMS locally, you'll to need run a production build of the site:

```
$ yarn build
$ yarn serve
```
